# Node JS Blog Rest APIs

A simple example of Blog Rest APIs using Node Js, Express and MongoDb.

## What you will get in this

- API's for User Authentication - Login, Logout, Change Password, Reset Password.
- API's to Add, Update and Delete existing Posts.

## Routes

- GET /posts        - To get all posts.
- POST /posts/add   - To add a post.
- GET /posts/:id    - To get details of a post by id.
- PUT /posts/:id    - To update existing post with id and specified parameters.
- Delete /posts/:id - To delete existing post.
 
# How to run the app

- Git clone repository
- Then run these commands in terminal/shell

```npm
    npm i
    npm start
```
